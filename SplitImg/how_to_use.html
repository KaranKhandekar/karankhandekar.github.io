SortImg Program Guide

How to Use SortImg

Step 1: Select Folder Location

Click the "Select Folder Location" button to choose the location where the white_background and non_white_background folders will be created.

Step 2: Run the Sorting Process

Click the "Run" button to select the source folder containing the images you want to sort. This will start the sorting process.

Step 3: Wait for Completion

Wait for the program to process and move the images. The status will be updated in the UI, and a message box will notify you once the process is complete.

Step 4: Check Sorted Images

Once the process is complete, check the white_background and non_white_background folders in the selected location for your sorted images.

How SortImg Works

The SortImg program sorts images based on their background color. Here's a breakdown of how it works:

Folder Creation

When you click "Select Folder Location", the program creates two folders in the chosen location:

  • white_background: For images with a white background
  • non_white_background: For images without a white background

Image Processing

The program processes each image in the selected source folder by checking the color of the top-right 5x5 pixel area. If all pixels in this area are white, the image is classified as having a white background; otherwise, it is classified as having a non-white background.

Moving Images

Based on the background color, images are moved to the respective folders:

  • If the background is white, the image is moved to the white_background folder.
  • If the background is not white, the image is moved to the non_white_background folder.

Completion Notification

Once all images are processed, a message box notifies you that the sorting is complete. The program also updates the status, count, and time taken for the process in the UI.
