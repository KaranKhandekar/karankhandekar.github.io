Whether you're a photography enthusiast, a designer, or just someone who wants to keep their image folders organized, SortImg simplifies the process.

Features

  1. Effortless Sorting: SortImg automatically categorizes images based on their background colors, saving you time and effort.
  2. Intuitive Interface: With its user-friendly interface, SortImg is easy to navigate, even for beginners.
  3. Quick Results: SortImg quickly processes your images, providing you with sorted folders in no time.
  4. Apple Silicon Chip Compatibility: SortImg is optimized for Apple Silicon chips, ensuring smooth performance on compatible devices. (Note: SortImg is not compatible with Apple Intel chips.)

Documentation: SortImg

1) How to use the app?
SortImg is a simple yet powerful tool designed to help users sort images based on their background colors. Follow these steps to use the app effectively:

  • Step 1: Launch the SortImg application.
  • Step 2: Click on the "Select Source Folder" button to choose the folder containing the images you want to sort.
  • Step 3: Once you've selected the source folder, click on the "Run" button to start the sorting process.
  • Step 4: Wait for the app to process the images. You'll see the status of the process displayed on the interface.
  • Step 5: After the sorting is complete, you'll receive a notification indicating that the process is finished.
  • Step 6: Navigate to the destination folders (white_background and non_white_background) to view the sorted images.

2) What is this app all about?
SortImg is a utility tool developed to simplify the task of organizing image collections. It automates the process of sorting images based on their background colors, saving users valuable time and effort. Whether you're managing a large library of photos or sorting through design assets, SortImg provides a convenient solution to streamline the workflow.

3) How it is built?
SortImg is built using Python programming language and utilizes several libraries and frameworks:

  • Tkinter: Tkinter is used to create the graphical user interface (GUI) of the application, providing users with an intuitive interface to interact with.
  • Pillow (Python Imaging Library): Pillow is used for image processing tasks, such as loading images, analyzing their background colors, and sorting them into appropriate folders.
  • OS and shutil modules: These built-in Python modules are used for file handling and directory operations, allowing SortImg to create destination folders and move sorted images accordingly.
  • Time module: The time module is used to measure the time taken for the sorting process, providing users with information on the processing duration.
  • Customtkinter library: Customtkinter library is used to customize the appearance of GUI elements, enhancing the visual appeal of the application.

SortImg is designed to be efficient, reliable, and easy to use, making it an essential tool for anyone dealing with image organization tasks.
