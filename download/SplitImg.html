Split Image

SplitImg for Designers is a utility application developed for Saks Global. This tool automates the process of organizing, tagging, and distributing images among designers. It assigns color tags based on the background color of the images and ensures a smooth workflow by splitting images into designated folders for each designer.

Key Features

  • Supports multiple image formats: PNG, JPEG, BMP, GIF, TIFF
  • Automatically assigns green or yellow tags based on the image's background color
  • Distributes images among selected designers using a round-robin method
  • Displays the number of images processed and the time taken for the task
  • Simple UI for quick and efficient image sorting

How It Works

SplitImg works by processing images in a selected folder and distributing them across multiple designer folders. The software performs the following steps:

  1. Image Identification: The software scans the source folder for images in supported formats (PNG, JPEG, BMP, GIF, TIFF).
  2. Background Color Check: Each image's top-right corner is analyzed to determine if the background is white or gray. If the background is white, the image is tagged with a yellow label; if gray, it is tagged with a green label.
  3. Round-Robin Distribution: The images are distributed among the specified number of designers using a round-robin method, ensuring an even split of images.
  4. Tagging: Tags (green or yellow) are applied to the images using macOS AppleScript commands. This allows easy identification of images based on background color.
  5. Logging and Feedback: The software keeps track of the number of images processed and the time taken for the operation, providing real-time feedback in the user interface.

Background Calculations

The software checks the background of each image by analyzing a small region of the image. Specifically, it looks at the top-right corner of the image (the last few pixels in that region). The following logic is applied:

  • If the pixels in the top-right corner match the color (255, 255, 255), which represents white, the image is considered to have a white background.
  • If the pixel color is close to RGB (128, 128, 128), the image is considered to have a gray background.
  • Images that don't fit into these categories are skipped, and the software doesn't apply any tags to them.

How to Use

Follow these simple steps to use the SplitImg software:

  • Step 1: Open the application and select the number of designers from the dropdown (up to 15). This will determine how the images are distributed.
  • Step 2: Click the "Run" button to begin the process.
  • Step 3: A file dialog will appear. Select the folder containing the images you want to process.
  • Step 4: The software will begin processing the images. As it works, it will display the number of images processed and the time taken for the task.
  • Step 5: Once the process is complete, you will be notified, and the images will be placed in separate folders for each designer, with appropriate tags applied.

Documentation: SortImg

1) How to use the app?
SortImg is a simple yet powerful tool designed to help users sort images based on their background colors. Follow these steps to use the app effectively:

  • Step 1: Launch the SortImg application.
  • Step 2: Click on the "Select Source Folder" button to choose the folder containing the images you want to sort.
  • Step 3: Once you've selected the source folder, click on the "Run" button to start the sorting process.
  • Step 4: Wait for the app to process the images. You'll see the status of the process displayed on the interface.
  • Step 5: After the sorting is complete, you'll receive a notification indicating that the process is finished.
  • Step 6: Navigate to the destination folders (white_background and non_white_background) to view the sorted images.

2) What is this app all about?
SortImg is a utility tool developed to simplify the task of organizing image collections. It automates the process of sorting images based on their background colors, saving users valuable time and effort. Whether you're managing a large library of photos or sorting through design assets, SortImg provides a convenient solution to streamline the workflow.

3) How it is built?
SortImg is built using Python programming language and utilizes several libraries and frameworks:

  • Tkinter: Tkinter is used to create the graphical user interface (GUI) of the application, providing users with an intuitive interface to interact with.
  • Pillow (Python Imaging Library): Pillow is used for image processing tasks, such as loading images, analyzing their background colors, and sorting them into appropriate folders.
  • OS and shutil modules: These built-in Python modules are used for file handling and directory operations, allowing SortImg to create destination folders and move sorted images accordingly.
  • Time module: The time module is used to measure the time taken for the sorting process, providing users with information on the processing duration.
  • Customtkinter library: Customtkinter library is used to customize the appearance of GUI elements, enhancing the visual appeal of the application.

SortImg is designed to be efficient, reliable, and easy to use, making it an essential tool for anyone dealing with image organization tasks.
